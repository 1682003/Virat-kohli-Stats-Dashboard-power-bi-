This project presents an interactive Power BI dashboard built using cleaned match statistics of Virat Kohli. The dashboard highlights his batting performance across years and formats and provides clear visual insights into his consistency and achievements.

Project Overview

The data was processed and structured to analyze year-wise and format-wise trends. The dashboard allows users to explore his performance in different match formats and seasons with interactive filters.

Key Features

• Year-wise and format-wise performance analysis
• Visual trends for runs, average, and strike rate
• Overview of centuries, half-centuries, and milestones
• Match-wise performance breakdown
• Interactive slicers for flexible exploration

Tools Used

Power BI
Excel or CSV data source

Outcome

The dashboard provides a clear statistical view of Virat Kohli’s batting performance, helping users understand patterns, trends, and overall career progression.
